# Five Nights at Gold's (FNAG)

A FNAF-inspired trading-themed horror survival game built with React + Tone.js.

## Features
- 12 rooms, 9 animatronics with unique behaviors
- Procedural audio via Tone.js
- Normal, Nightmare, Infinite, and Custom game modes
- Shop/upgrade system, achievements, and persistent save data
- Mini-game challenges (math, typing, memory, wire-cutting, etc.)
- Market events (Flash Crash, Circuit Breaker, Rate Hike, etc.)

## Run Locally
```bash
npm install
npm run dev
```

## Deploy
```bash
npm run build
npm start
```
